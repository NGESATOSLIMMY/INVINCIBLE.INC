#  INVINCIBLE.INC

##  Project Description

INVINCIBLE.INC is a web-based community reporting platform that allows users to anonymously report incidents affecting their society.

Users can submit reports about issues such as:
- Corruption
- Police brutality
- Poor infrastructure
- Health system failures
- Education problems

Each report can include:
- Title
- Category
- Location
- Description
- Image or video evidence (optional)
- Urgency level

Users can also:
-  Applaud reports they support
-  Comment on reports using anonymous usernames
-  View exposed (serious) cases
-  Browse all community reports

All data is stored locally using **localStorage**, meaning no backend server is required.
